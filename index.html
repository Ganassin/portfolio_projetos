    Projeto de Estudo 2: Desenvolvimento de Dashboard Gerencial para Marketplace de Restaurantes com Streamlit

    Este projeto foi desenvolvido para uma empresa fictícia, um marketplace de restaurantes, com o objetivo de criar um painel gerencial que auxiliasse o CEO a tomar decisões estratégicas baseadas em dados.
    Utilizando técnicas de análise de dados com Python e ferramentas como Streamlit e GitHub, o painel responde a perguntas chave relacionadas ao número de restaurantes, tipos de culinária, avaliações e diversas métricas por país e cidade.
