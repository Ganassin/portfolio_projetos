Projetos de Análise de Dados

Projeto HackDay Comunidade DS: Extração, tratamento e análise de Dados para Estratégias de Vendas no Mercado Editorial

Projeto desenvolvido durante um Hackday promovido pela Comunidade de Data Science (CDS) e teve como objetivo analisar dados do mercado editorial para gerar insights estratégicos sobre vendas de livros.
Foram aplicadas técnicas de SQL para extração e transformação dos dados, seguidas por manipulação e limpeza utilizando Python (Pandas, SQLAlchemy), onde as informações foram organizadas e exportadas para arquivos .csv, permitindo a construção de um dashboard interativo no Power BI.
Os insights obtidos revelaram padrões de demanda por gênero e formato de livro, auxiliando na tomada de decisões estratégicas, como a priorização de determinados formatos e preços mais competitivos. Além disso, foram identificadas oportunidades para aumentar a divulgação de livros bem avaliados, mas com baixo faturamento.

Ferramentas Utilizadas:

  • Python (Pandas, SQLAlchemy)
  • SQL
  • Power Bi
  • Git/GitHub
  • GitHub

Desenvolvimento de um Painel Gerencial para Negócio com o Stremlit.

Desenvolvimento de Dashboard Gerencial para Marketplace de Restaurantes com Streamlit

Este projeto foi desenvolvido para uma empresa fictícia, um marketplace de restaurantes, com o objetivo de criar um painel gerencial que auxiliasse o CEO a tomar decisões estratégicas baseadas em dados.
Utilizando técnicas de análise de dados com Python e ferramentas como Streamlit e GitHub, o painel responde a perguntas chave relacionadas ao número de restaurantes, tipos de culinária, avaliações e diversas métricas por país e cidade.
O resultado é um dashboard interativo e intuitivo, acessível via web, que facilita a visualização de insights críticos sobre a performance do negócio, como os países com os melhores restaurantes e os tipos de culinária mais populares.

Ferramentas Utilizadas:

  • Python (Pandas, Matplotlib, Plotly, Folium)
  • Jupyter Lab
  • Terminal
  • Streamlit
  • Streamlit Cloud
  • GitHub
  • GitHub
