Projeto de Estudo 1: Desenvolvimento de um Painel Gerencial para Negócio com o Stremlit.

Nesse projeto, os conceitos de Programação em Python, manipulação de dados, pensamento estratégico e lógica de negócio, junto com ferramentas de desenvolvimetno web como o Streamlit e Github, foram usados para desenvolver um painel gerencial com as principais métricas de uma empresa marketplace de delivery de comida na India.

Projeto de Estudo 2: Desenvolvimento de Dashboard Gerencial para Marketplace de Restaurantes com Streamlit

Este projeto foi desenvolvido para uma empresa fictícia, um marketplace de restaurantes, com o objetivo de criar um painel gerencial que auxiliasse o CEO a tomar decisões estratégicas baseadas em dados.
Utilizando técnicas de análise de dados com Python e ferramentas como Streamlit e GitHub, o painel responde a perguntas chave relacionadas ao número de restaurantes, tipos de culinária, avaliações e diversas métricas por país e cidade.
O resultado é um dashboard interativo e intuitivo, acessível via web, que facilita a visualização de insights críticos sobre a performance do negócio, como os países com os melhores restaurantes e os tipos de culinária mais populares.

Ferramentas Utilizadas:

  • Python (Pandas, Matplotlib, Plotly, Folium)
  • Jupyter Lab
  • Terminal
  • Streamlit
